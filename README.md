##  Projects
### 🤖 Application of Neural Networks in Sign Language to Speech Translation Systems [Master Thesis @ Technical University of Lodz]
The master’s thesis addresses the problem of automatic sign language recognition using
neural networks. The primary goal of the study is to design and evaluate the architectures of
machine learning models based on convolutional neural networks and graph neural networks.
The project consists of data preparing and transforming stage, designing the architecture of the
models, the process of training and evaluating the models using the chosen metrics.
Additionally, the study focused on explainable artificial intelligence to try and understand the
reasoning behind the decisions made by trained models.

Key words: American Sign Language (ASL), Polish Sign Language (PJM), Sign Language
Recognition, Neural Networks, Convolutional Neural Networks, Graph Neural Networks,
Explainable Artificial Intelligence

The Artificial Intelligence model was prepared using @PyTorch.

### 🥕 [FitYouFood](https://github.com/1anKoz/FitYouFood)
Group project – part of the 1st year on Technical University of Lodz.

An application that can be used to track workouts, suggesting execrcises and generating dietary plans.

**My contribution** consisted of developing the web application in React and preparing CI/CD pipelines.

Application architecture:
- **backend** - based on **.NET8** with Microsoft SQL Server database,
- **frontend** - progressive web application built using **React**,
- **AI** - never finished, model architecture prepared using **PyTorch**,
- **DevOps** - CI/CD pipeline using GitHub Actions.

The whole application can be run using Docker.
